# react-ssr-setup

Simple nodejs/react boilerplate for setting up server-side rendering

## Quickstart
- run `npm install` to install the dependencies
- run `npm start` to start the application on port `1234`

## Production
- run `npm install` to install the dependencies
- run `NODE_ENV=production npm run build` to build the relevant bundles
- copy `app.js` and the `public` dir where you want to run the application
- run `node app.js` and the app will start

## Configuration
- you can export the `PORT` environment variable to change the port
